# Responsive Admin Dashboard

A modern and responsive admin dashboard built using only HTML and CSS. This project focuses on creating a clean user interface with responsive layouts, reusable components, and smooth hover effects without using any JavaScript or CSS frameworks.

## Features

* Responsive layout using CSS Grid and Flexbox
* Dashboard summary cards
* Sidebar navigation
* Search bar
* User profile section
* Project status cards
* Recent activity feed
* CSS Custom Properties (Variables)
* Hover animations and transitions
* Mobile-friendly design using media queries

## Technologies Used

* HTML5
* CSS3

  * Flexbox
  * CSS Grid
  * CSS Variables
  * Media Queries
  * Transitions
